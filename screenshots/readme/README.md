# README Screenshots (refreshed)

Fresh, **unframed** re-captures of the views shown in the top-level `README.md`,
seeded with current demo data and a frozen mid-morning clock (so the day reads
as *in progress*, not overdue). Apply your usual device-frame / background
treatment, then swap these into `README.md` in place of the dated originals.

Filenames match the originals in `../` (the parent `screenshots/` folder) so they
drop straight in.

```bash

```
